# Technical-safe checklist

Before finalizing a humanized technical rewrite, verify:

- commands unchanged
- flags unchanged
- code blocks unchanged unless asked
- paths and filenames unchanged
- URLs unchanged
- numbers, thresholds, units, dates, and versions unchanged
- `must/should/may` semantics unchanged
- technical terms not replaced with weaker synonyms
- headings kept if they improve scanning
- fluff removed without removing meaning
- no fake warmth, hype, or vague promises added
